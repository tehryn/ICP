Detailed Description

Class representing table with cards. Methods represents all possible moves, player can do. <font color="red">DO NOT</font> use object of this class until you call method new_game() or load_game(String filename).
There are seven working stacks, IDs from 0 to 6,
four color stacks, IDs from 0 to 3,
one visible deck and one hidden deck.
Object of Board also holds history of moves and current score.

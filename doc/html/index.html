Installation

To install application, you need to have QT 5 installated on your computer (with qmake).
Since you have qmake and QT 5 with qmake, you can run "make" from root directory of project.
Binnary files are stored in src directory (./src/hra2017 and ./src/hra2017-cli). You can run both applications with "make run"
Enjoy the game and have fun!
